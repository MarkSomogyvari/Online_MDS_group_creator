# Contributing

Thank you for your interest in contributing to Workshop Group Finder! This project follows these guidelines:

## Development Workflow

1. **Log every development step** in the project documentation
2. **Use git for version control** - commit changes with clear, descriptive messages
3. **Test changes** before committing
4. **Update documentation** as you add new features

## How to Contribute

1. Fork the repository
2. Create a new branch for your feature/bugfix
3. Make your changes following the coding style
4. Test your changes
5. Commit your changes with clear messages
6. Push to your branch
7. Open a Pull Request
